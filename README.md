# rakuen

[![CircleCI](https://circleci.com/gh/TerrorJack/rakuen/tree/master.svg?style=shield)](https://circleci.com/gh/TerrorJack/rakuen)
[![Docker Pulls](https://img.shields.io/docker/pulls/terrorjack/rakuen.svg)](https://hub.docker.com/r/terrorjack/rakuen/)

A build environment for my personal Haskell projects.

* Uses the latest Ubuntu release & [toolchain](https://launchpad.net/~ubuntu-toolchain-r/+archive/ubuntu/test)
* Based on [`stack`](https://docs.haskellstack.org/en/stable/README/) and [Stackage Nightly](https://www.stackage.org/nightly)
* Pre-installs a bunch of packages. Selection of packages is super opinionated.
